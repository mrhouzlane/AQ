## Development

### web

#### setup

Need to build contract first

```
npm i
npm run build
```

Then, install in web directory

```
// cd web
npm i
```

#### development

```
npm run dev
```

#### build

```
npm run build
```

you can upload web/build to ipfs for hosting.

### Test

```
npm test
```
